## 1. 방한 외국인 2,000만 시대의 정량 지표

### 1.1 연도별 방한객 추이
- 2024년: 1,636만 명 (한국관광공사 공식 통계)
- 2025년 잠정: 약 1,893만 명
- **2026년 전망: 최대 2,126만 명** (야놀자리서치 LSTM 예측)
- 한국관광공사 2026.1 업무보고: **"2030년 방한 3,000만 조기 달성"** 공식 선언

### 1.2 방문 목적의 변화
2024 외래관광객 실태조사 (한국문화관광연구원, kcti.re.kr):
- **미식 탐방 희망률: 68%** (전년 대비 +11%p)
- 쇼핑 희망률: 51.5% (전년 대비 -8%p)
- **미식이 쇼핑을 역전한 최초 연도**

### 1.3 재방문자 비율
- 3년 내 3회 이상 재방문자: **45%** (크리에이트립 2026.02)
- 방문 계기 1위: **한류 콘텐츠 접촉 38.3%**
- 대만·홍콩·일본 재방문율 특히 높음

### 1.4 지역 집중
- **서울 방문율: 80%** (양문석 의원 2024 국감 질의)
- 명동 2025년 상반기 방문 450만 명
- 명동 파출소 업무의 **70~80%가 외국인 응대**

## 3. 메뉴 오역 실태 (공식 조사)

**문화체육관광부·한국관광공사 공동조사** (보도자료 mcst.go.kr pSeq=15387):

오역 발생 원인 비중:
- 간판·광고업체: **41.8%**
- 프랜차이즈 본사: 27.5%
- 업주 자체 번역: 18.3%
- 기타: 12.4%

**핵심 인사이트**: 오역의 약 70%가 전문 번역자 없이 기계번역·자동번역을 거쳤다. 공식 표준 번역 DB(한식진흥원 800선)가 있음에도 실무 현장에 침투하지 못한 구조. → **MenuLens가 기술로 해결 가능한 영역**.

대표적 오역 사례:
- "육회" → "6 times"
- "동태찌개" → "Dynamic Stew"
- "삼겹살" → "three layers of flesh"

## 4. 기존 솔루션의 구조적 한계

### 4.1 Visit Korea 공식앱
- 구글 플레이 평점: **3.1/5.0** (리뷰 분석 결과)
- 주요 불만: "Check your connection" 오류, 7일 플래너 제한, UI 구식
- 외국인 대상 앱 사용자 만족도 조사: **"가장 불만족 앱 1위 (30.2%)"**

### 4.2 Google Maps
- 한국 지도 데이터 반출 3차 거절 (2016, 2022, 2023)
- **도보 길찾기 불가** — 뉴욕타임즈·CNN 2025년 보도
- CNN 2025.9: "Why doesn't Google Maps work in South Korea, one of Asia's most developed countries?"

### 4.3 Papago
- 한국어 카메라 번역 품질 우수하나 **번역만** 제공
- 가격·알레르기·문화 정보 0
- 모든 리뷰어(Trip.com, Korealocally 등)의 공통 결론: "Install both Papago and Google Translate"

### 4.4 글로벌 AI 여행 서비스
- **Mindtrip, Layla, Roam Around**: TourAPI 미연동, 한국 공공데이터 0
- **Perplexity**: 2025 Columbia Journalism Review 감사 결과 **환각 오류율 37%**
- **ChatGPT Travel**: 실시간 데이터 부재, "존재하지 않는 호텔 추천" 다수 보고

### 4.5 예약 앱 접근성
- 캐치테이블·테이블링·포잉: **한국 전화번호 본인인증 필수**
- 2026.3 서울시-캐치테이블 MOU로 일부 완화되었지만 **글로벌 버전은 국내 매장의 1/5만 커버**

## 5. TourAPI 활용도 분석 (13회 공모전 역사)

### 5.1 공모전 규모 (2025년 기준)
- 접수팀: 156~200팀 (연도별)
- 수상: 60팀 (대상1, 최우수6, 우수13, 장려40)
- 상위 7팀 당선률: **약 4~5%**

### 5.2 역대 수상작의 TourAPI 활용 패턴
- **모든 수상작이 TourAPI 필수 활용** (규정)
- 주로 사용된 Operation: `areaBasedList`, `detailCommon`, `searchKeyword` (기본 3종)
- **TourAPI LOD SPARQL 엔드포인트 활용 수상작: 0건** (13년간)
- 다국어 API 활용은 2022년 해외서비스 부문 신설 후 증가

### 5.3 수상 패턴 분석
- 2022 대상 "나만의 서울관광": 간결·시각 중심
- 2023 대상 "도토리": 지역특화 단순 컨셉
- 2024 대상 '싸이클로라마': 자전거 라이딩 특화 (한 가지만 잘 함)
- 2024 최우수 SPOT!: GitHub 공개된 MVP, 실제 작동

**공통점**: "한 가지에 집중 + 실제 작동"

### 5.4 외국인 타깃 수상작
- 2022 AboutKorea (해시태그 K-컬처) — 유일한 외국인 중심 우수상
- 그 외 대부분은 국내 사용자 중심
- **"외국인 × AI × 3중 검증" 조합 수상작 0건**

## 6. 공공데이터 생태계

### 6.1 활용 가능 데이터셋 (MenuLens 기준)

| 데이터 | 소관 | 개방 형태 | 상태 |
|---|---|---|---|
| TourAPI 4.0 | KTO | Open API (15101578 외 5종 다국어) | 운영 중 |
| TourAPI LOD | KTO | SPARQL Endpoint | 운영 중, 미활용 |
| 한식진흥원 800선 | 한식진흥원 | 파일 (15129784) | 2023.06 업데이트 |
| 한국소비자원 참가격 | 한국소비자원 | 포털 + API 3043385 | 매주 갱신 |
| 식약처 식품영양성분DB | 식약처 | API 15127578 | 운영 중 |
| 외래객 친화 관광정보 | KTO | API 15111159 | 운영 중 |
| 부산관광공사 관광 공공데이터 | 부산관광공사 | 45종 (2025.09 전면 개방) | 신규 |
| KMF 할랄 인증 | 한국이슬람교중앙회 | 홈페이지 (API 미제공) | 수동 수집 |
| 카카오 로컬 API | 카카오 | REST API (일 10만 건 무료) | 상용 |

### 6.2 MenuLens의 데이터 교차 가치
6개 공공데이터를 통합해야만 가능한 기능:
- **공식 표준 번역**: TourAPI 다국어 + 한식진흥원 800선
- **가격 적정성**: 참가격 + 카카오 로컬 + TourAPI 식당 정보
- **알레르기 판정**: 식약처 + 한식진흥원 재료 정보 + LLM
- **특수 식단**: KMF 할랄 + 식품영양성분

- OpenAI Swarm, CrewAI 상용화

### 7.4 오디오 TTS
- Google Cloud TTS: 한국어 자연음성, 월 4M chars 무료

## 8. 타이밍 삼각형

**핵심 인사이트**: 2026년은 세 가지가 **처음으로 동시에 성립한 해**.

```
        수요 정점 (방한 2000만)
               ▲
              / \
             /   \
            /     \
    AI 기술 ──── 공공데이터
    성숙도       개방도
   (멀티모달)   (800선+참가격+LOD)
   
        │
        ▼
    MenuLens 가능 창
```

세 꼭짓점이 모두 올해 처음 교차:
- 한식진흥원 800선 개방: 2023
- TourAPI LOD 안정화: 2024
- 방한 2000만 돌파: 2026
- Gemini Flash Vision 대중화: 2025
- 바가지 국가 어젠다화: 2025~2026

이 삼각형이 **다음 5년 내 다시 이 상태로 정렬될 확률은 낮다**. 타이밍이 작품의 일부인 이유.
