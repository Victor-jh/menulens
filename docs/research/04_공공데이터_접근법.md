## 1. TourAPI (필수)

### 접근
- **공공데이터포털**: https://www.data.go.kr/data/15101578/openapi.do (국문)
  - 영문: 15101753, 일문: 15101771, 중문 간체: 15101793, 중문 번체: 15101792
- **한국관광 콘텐츠랩**: https://api.visitkorea.or.kr
- **인증키**: `data.go.kr` 활용신청 → 개발계정 → 인증키 확인
- **LOD SPARQL**: http://data.visitkorea.or.kr/sparql (별도 인증 불필요)

### 주요 Operations (KorService2)
| Op | 파라미터 | 반환 |
|---|---|---|
| `areaBasedList2` | `contentTypeId=39` (음식점), `mapX`, `mapY`, `radius` | 식당 리스트 |
| `detailCommon2` | `contentId` | 기본 정보 (이름·주소·전화·홈페이지) |
| `detailIntro2` | `contentId`, `contentTypeId=39` | 대표메뉴, 영업시간, 휴무일, 좌석수 |
| `detailInfo2` | `contentId` | 반복 정보 |
| `detailImage2` | `contentId` | 대표 이미지 |
| `locationBasedList2` | `mapX`, `mapY`, `radius` | 현재 위치 기반 |
| `searchKeyword2` | `keyword`, `contentTypeId` | 키워드 검색 |

### 한계
- 음식점 `detailIntro2`의 `firstmenu` 필드는 대표 메뉴 **이름만** 제공. **가격 없음**
- 좌표 오차: 구도심 ±5m, 지방 ±50m
- 다국어 번역 품질: 영문 양호, 중일 기계번역 흔적 있음 → **한식진흥원 800선으로 교정**
- LOD SPARQL 응답 지연 5~15초, 동시 요청 5개 이상 타임아웃 → ETL로 로컬 Neo4j/pgvector 복제 필요

### D1 액션
- [ ] data.go.kr 로그인
- [ ] 한국관광공사_국문 관광정보 서비스_GW (15101578) 활용신청
- [ ] 한국관광공사_영문 관광정보서비스_GW (15101753) 활용신청
- [ ] 한국관광공사_일문 관광정보서비스_GW (15101771) 활용신청
- [ ] 한국관광공사_중문간체 관광정보서비스_GW (15101793) 활용신청
- [ ] 한국관광 콘텐츠랩 (api.visitkorea.or.kr) 회원가입 (공모전 접수 필수)

## 2. 한식진흥원 길라잡이 800선

### 접근
- **공공데이터포털**: https://www.data.go.kr/data/15129784/fileData.do
- **포맷**: 파일 데이터셋 (Excel/CSV)
- **내용**: 800개 메뉴, 영·일·중 표준 번역 + 설명

### 활용
```python
# D3에서 이 스크립트로 적재
import pandas as pd
df = pd.read_excel("backend/data/hansik_800.xlsx")
# 컬럼: 메뉴명, 영문, 일문, 중문, 설명, 분류

# 각 메뉴명 임베딩
from openai import OpenAI
client = OpenAI()
for row in df.iterrows():
    emb = client.embeddings.create(
        model="text-embedding-3-small",
        input=row['메뉴명']
    ).data[0].embedding
    # Supabase pgvector에 upsert
```

### 한계
- 800개는 **대표 한식**. 외국인이 자주 만나는 변형 메뉴(얼큰한 김치찌개, 해물 된장)는 fuzzy match 필요
- 업데이트 주기: 최신 2023.06

### D1 액션
- [ ] 파일 다운로드 → `backend/data/hansik_800.xlsx`

## 3. 한국소비자원 참가격 (외식비)

### 접근
- **포털**: https://www.price.go.kr/tprice/portal/servicepriceinfo/dineoutprice/dineOutPriceList.do
- **API**: data.go.kr `3043385` (생필품 가격)
- **외식비**: 웹 포털에서 조회 (API 미제공), 수동 스냅샷 후 JSON 저장

### 8품목 (서울 기준, 2025-12)

CONSUMER_PRICE_SEOUL_2025 = {

    "칼국수": 9846,

    "삼겹살": 21500,        # 200g 기준
    "비빔밥": 11200,

### 한계
- 매월 갱신이나 API 없음 → 수동 스냅샷 필요
- 서울 기준. 부산·경북 등 지역 편차는 보정 필요 (Phase 2)
- 8개만 커버 → 그 외 메뉴는 Claude 상식 추론 + "신뢰도 낮음" 배지
