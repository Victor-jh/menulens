## 0. 왜 맞춤 메뉴판이 필요한가

실제 식당 메뉴판 촬영은 **3개 제약**을 동시에 만족해야 한다:

1. **초상권·간판 노출 금지** (상호·영업주 식별 가능 시 추후 분쟁 소지)
2. **색깔 3종 결과가 자연스럽게 나오는 메뉴 구성**이 우연히 맞아떨어져야 함 (🟢·🟡·🔴 각 1~2건씩)
3. **판독 용이한 레이아웃** — 카메라가 한 프레임에 전체 메뉴를 잡을 수 있는 크기·폰트·대비

이 3개를 모두 충족하는 실제 메뉴판을 찾는 건 우연에 기대는 것. **맞춤 인쇄본**이 D7 Hard Gate 직후 Plan A/B 모두에서 안정적 촬영 재료가 된다.

## 1. 메뉴 구성 (색깔 3종 자연 노출 설계)

**설계 의도**:
- 🟢 3건 (1·2·3): 주문 가능한 선택지로 충분히 보이는 수
- 🟡 2~3건 (4·5·7): 경계 케이스로 "약간 비쌈"·"주의" 레이블 설득력
- 🔴 1건 (6): 바가지 경보의 명확한 예시
- ⚪ 1건 (8): 참가격 외 메뉴는 정직하게 "정보 없음" — 시스템 신뢰성 증명

**참고**: 위 참가격 평균가는 백엔드 측에 실제 적재된 값(price.go.kr 2025.12 스냅샷 기준)을 반영. 촬영 시 실제 시스템 판정 결과와 동일해야 함.

## 2. 레이아웃 사양

### 2.1 용지
- **A3 (297×420mm) 세로 방향** — 카메라로 1프레임에 잡히는 최대 크기
- **120gsm 이상 매트 코팅** — 조명 반사 최소화
- 단면 컬러 인쇄

### 2.2 폰트·크기
- **상단 제목**: "미정(가상 식당명)" — 시연용 임의 상호, 실제 존재 상호와 겹치지 않도록
  - 권장: "산뜻한 한식당" / "Sandeut Korean Kitchen" 등 일반명
  - 폰트: 나눔명조 36pt Bold
- **메뉴명 (한글)**: 나눔고딕 24pt
- **메뉴명 (영문)**: Arial 14pt
- **가격**: 나눔고딕 20pt Bold (오른쪽 정렬)
- **설명 한두 줄 (옵션)**: 나눔고딕 10pt (생략 가능 — Menu Reader가 읽어들일 필요 없음)

### 2.4 색상·대비
- 배경: 아이보리(#F5F1E8) 또는 순백(#FFFFFF)
- 글자: 짙은 회색(#333333) 또는 검정
- 가격: 살짝 붉은 톤 또는 검정 (강조 자제 — AR 오버레이 색과 충돌 방지)
- **절대 피할 것**: 메뉴판 자체에 녹색·빨간색·노란색 배경·테두리 사용 (AR 색깔 오버레이와 시각 충돌)

### 2.5 실제 생산
1. 본 문서를 기반으로 Canva 또는 Google Slides에서 A3 문서 작성
2. PNG 300dpi 또는 PDF로 export
3. 사무실 프린터 or 로컬 인쇄소에서 A3 컬러 단면 인쇄 (건당 ~2,000원)
4. 예비 3장 인쇄 (촬영 중 손상·리테이크 대비)

## 3. 촬영 시 활용 시나리오

### 3.1 Plan A (실제 식당 섭외 성공 시)
- 실제 식당 메뉴판을 **기본 프레임**으로 사용
- **단, 바가지·🔴 판정이 실제 식당 메뉴에서 안 나올 경우**에만 이 샘플 메뉴판을 보조로 겹쳐서 시연 (1~2컷)

### 3.2 Plan B (스튜디오 촬영)
- 본 샘플 메뉴판을 **전체 영상에서 유일한 메뉴판**으로 사용
- 배경: 원목 테이블 + 수저·물컵 1개 + 조명
- 카메라 각도: 약간 위에서(30도) 내려다보며 메뉴판 전체가 프레임에 들어오도록
