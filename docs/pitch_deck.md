## 2. 섹션별 스크립트 상세

### [0:00~0:10] Hook — "4초"

**영상 연출**:
- 첫 프레임: 외국인 인물 눈 클로즈업, 흑백 톤 → 컬러 전환
- 배경 흐리게 (bokeh), 인물에 포커스
- 메뉴판은 인물 어깨 너머로 살짝만 보이도록

**내레이션 (여성 차분한 톤, 영어 primary)**:
> "4 seconds. That's how long tourists stay in front of a Korean menu before turning away."

**내레이션 (한국어 보이스오버, 자막 동기화)**:
> "단 4초. 그 시간이면 외국인 관광객의 결정은 끝난다."

**연출 의도**: 통계보다 **감정적 순간**으로 시작. 심사위원의 공감대 획득.

---

### [0:10~0:20] Problem — 4가지 질문

**영상 연출**:
- 메뉴판이 중앙 → 4개 질문 풍선이 사방에서 팝업
- 각 질문 0.5초 등장 → 2초 유지 → 사라짐
- 질문들이 "폭격"처럼 동시다발로 뜨는 느낌

**자막 레이아웃**:
```
    "What is this dish?"
         이게 뭐죠?

"Is this price fair?"           "Can I eat this?"
   이 가격이 맞나요?               먹을 수 있나요?

    "How do I order?"
      어떻게 주문하죠?
```

**내레이션 (옵션, 자막만 써도 OK)**:
> "Every year, 20 million tourists face this moment. Papago translates — but nothing tells you *what to do*."

---

### [0:20~0:30] Solution — Statement + Shutter

**영상 연출**:
- 화면 전환: 흐린 메뉴판 → 선명한 휴대폰 카메라 뷰
- 휴대폰이 메뉴판에 가까워짐 (Dolly-in)
- "찰칵" 셔터음과 동시에 화면 전체 0.1초 플래시
- 직후 3초짜리 원형 로딩 애니메이션 (컬러: MenuLens 브랜드 그린)

**자막 (큰 폰트, 중앙 배치)**:
```
[0:20~0:23]
  DON'T READ.
  읽지 마세요.

[0:23~0:30]
  JUST ORDER WHAT'S GREEN.
  초록색만 주문하세요.
```

**오디오**:
- 0:25: 셔터음 SFX (아이폰 기본음 또는 Pixabay 무료)
- 0:25~0:28: 로딩 사운드 (저음 whoosh)
- 0:28: "딩" 알림음
